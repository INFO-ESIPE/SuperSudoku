// FR
Etape 1: éxécutez la commande "make" dans le répertoire "TP7_Vivien_Alphonse" pour compiler le programme.
Etape 2: éxécutez la commande "./Sudoku" avec comme argument le chemin vers le fichier contenant le sudoku à résoudre (ex: "./Sudoku data/grid3.txt") pour lancer le programme. (Il y a un fichier par défaut pour le cas où vous ne mettez pas d'argument)
Etape 3: A vous de jouer !
Remarque: Dans le fichier "main.h", vous pouvez modifier la taille de la grille (constante "SIZE_OF_GRID") à 9 pour une grille 9x9, 6 pour une grille 6x6, vous pouvez aussi modifier le titre (constante "TITLE"), la largeur (constante "WIDTH") et la hauteur (constante "HEIGHT") de la fenêtre.
Vous pouvez mettre la contante "SHOW_RIGHT_ANSWERS_IN_VALUE_SELECTOR_GRID" à 0 pour faire en sorte que la grille qui permet de sélectionner une valeur affiche les valeurs acceptées pour la case sélectionnée en vert et les valeurs non acceptées en rouge.

// EN
Step 1: run the command "make" in the "TP7_Vivien_Alphonse" directory to compile the program.
Step 2: run the command "./Sudoku" with the path to the file containing the sudoku to solve as an argument (ex: "./Sudoku data/grid3.txt") to launch the program. (There is a default file in case you don't put an argument)
Step 3: Have fun !
Remark: In the "main.h" file, you can modify the size of the grid (constant "SIZE_OF_GRID") to 9 for a 9x9 grid, 6 for a 6x6 grid, you can also modify the title (constant "TITLE"), the width (constant "WIDTH") and the height (constant "HEIGHT") of the window.
You can set the constant "SHOW_RIGHT_ANSWERS_IN_VALUE_SELECTOR_GRID" to 0 to make the value selector grid display the accepted values for the selected case in green and the non-accepted values in red.
